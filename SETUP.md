# Quick Start - Push to GitHub

## Step 1: Download the folder
Download the `meals-on-wheels-sms` folder from this chat

## Step 2: Push to GitHub

Open terminal in the folder and run:

```bash
git init
git add .
git commit -m "Initial project structure"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/meals-on-wheels-sms.git
git push -u origin main
```

## Step 3: Add Your Team
1. Go to your repo on GitHub
2. Settings → Collaborators → Add people
3. Send them the repo link

## Step 4: They Clone It
```bash
git clone https://github.com/YOUR-USERNAME/meals-on-wheels-sms.git
cd meals-on-wheels-sms
```

Done! Everyone can start adding files to their folders tomorrow.
